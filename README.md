## ⚙️ Configuring your exchange link

The tool needs permission to read your trade data. You must create an API key in your OKX account settings.

1. Log into your OKX account in your web browser.
2. Go to the API management section under your profile settings.
3. Select "Create V5 API Key".
4. Give your key a nickname.
5. Select the "Read" permission. You do not need to enable "Trade" or "Withdraw" permissions for monitoring purposes.
6. Copy your API Key, Secret Key, and Passphrase.
7. Open the monitoring tool on your computer.
8. Paste these credentials into the settings menu inside the application.
9. Click "Save" to apply your changes. 

The application uses these details to fetch real-time data from your exchange wallet. It never shares these keys with outside parties.

## 📈 Monitoring your grids

Once connected, the main screen displays your active ETH grid strategies. The dashboard updates every few seconds.

* Grid Status: Tells you if the grid remains active or if the price broke the set range.
* Profit and Loss: Shows gains or losses for the current cycle.
* Position Size: Shows the total ETH held in the current grid.
* Suggested Adjustments: Offers simple visual cues when you reach the upper or lower price bounds.

If the market price nears your grid limit, the application triggers a highlight color. This reminds you to pause or adjust the grid settings. You handle all updates through the OKX website while using this tool to monitor the data.

## 🛠 Troubleshooting common issues

If the application fails to fetch data, check your internet connection first. Ensure your API keys remain valid and have not expired. Some users experience issues with firewall settings. If the windows block the connection, add an exception for the monitor tool in your Windows Security settings.

If the application window feels slow, resize the window to refresh the layout. You can always restart the application if the connection stalls. The software keeps a local log file in the installation folder. You can open this file to see error details if the app reports an issue.

## 📖 Glossary

* Perpetual Contract: A crypto derivative that tracks the price of a specific coin without an expiration date.
* Grid Strategy: A trading method that buys at low prices and sells at high prices within a range.
* API Key: A unique code that allows software to talk to your exchange account.
* Passphrase: A security password needed alongside your API keys to verify your identity.
* Dashboard: The visual interface where you see your trade data.
